On the Web

Here are a few of the web development/content management systems projects I have worked on

Hidden Histories

Over the past year, I've had the opportunity to work with an amazing research team to develop a web application for the Hidden Histories project at NJIT. This web app is hosted via Azure, and communicates with containers hosted in Blob Storage. While my role was to develop the app in JavaScript, I had the opportunity to work with Unity developers, artists, and historians to create a cohesive user experience.

While the tool itself is still in the process of being made public, the project has a ReadMe on our hub page, and will hopefully answer any questions
